Detailed Description

Example application demonstrating the use of the XBee library.

This file contains a sample application that demonstrates how to use the XBee library to communicate with XBee modules in a platform-agnostic environment. It showcases basic operations such as initializing the XBee module, connecting to the network, and transmitting & receiving data.
